The hallucination problem in financial modeling AI

Rogo, Hebbia, and other LLM-first tools generate cells that contain values, not formulas. Numbers that drift from the underlying transaction. Cells that can't be traced back to a source. Wall Street Prep's 2026 benchmark ranked these tools below Claude + Excel — because rating-agency review requires every number to defend itself.

ModelForge takes a different architectural bet: LLMs produce specs + sources + narrative. Deterministic Python produces the workbook. The LLM never writes a number into a cell. A typed YAML spec drives an openpyxl builder. A 12-check QC gate validates before export. Every cell is queryable back to its source via a first-class linkage graph.

How it works

1

Spec or data room

Provide a typed YAML spec, or point ModelForge at a directory of PDFs and let it auto-extract. Every extracted number gets a source ID + page reference + confidence score.

2

Deterministic build

Python emits an openpyxl workbook with every cell live-formulated, color-coded by convention, named-ranged, and connected to the SQLite linkage graph.

3

QC gate + delivery

12 automated checks (BS ties, CFS reconciles, sign convention, formula ratio, source coverage). Export to PowerPoint + Word for committee. Re-run anytime: same spec → same workbook.

Install

As an MCP server (Claude Code, Cursor, Cline, ChatGPT Enterprise)

pip install "modelforge[mcp,export]"
modelforge-mcp  # stdio server — wire to your MCP client config

As a Python CLI

pip install modelforge
modelforge build examples/unitranche_cdmo.yaml
modelforge qc output/unitranche_cdmo.xlsx

Auto-ingest a data room (PDFs → YAML)

pip install "modelforge[ingest]"
export ANTHROPIC_API_KEY=sk-ant-...
modelforge ingest path/to/dataroom/ --template project_finance -o my_deal.yaml
modelforge build my_deal.yaml
modelforge qc output/my_deal.xlsx

Honest FAQ

Is this better than Rogo?

On output quality (formula discipline + source traceability + audit gate), yes — by a wide margin. On distribution, no — Rogo has scale and venture backing. ModelForge closes the distribution gap via MCP without raising capital. Pick it if your reviewers audit cells; pick Rogo if you need a Bloomberg-style sales motion.

Does it support US GAAP / UK FRS / German HGB?

IFRS 9 EIR is canonical and tested. Italian regulatory pack is native (IRES/IRAP/SIIQ/PEX, AIFMD II, L.130/1999, GACS, FER X). US GAAP variants are in v0.10 roadmap (Q3 2026). UK FRS 102 and German HGB queued for Q4 2026.

SOC 2 / ISO 27001?

Not yet. It's a tool you run locally or on-prem — your security perimeter is your own infrastructure. SOC 2 Type II is Phase B. For regulated deployments today, it runs on-prem with audit logs.

Live data feeds from Bloomberg / FactSet?

Not natively. CSV / XBRL / SEC EDGAR adapters exist. Bloomberg / FactSet / Capital IQ partnerships are Phase B (data licensing $150-300K/yr). For now, paste pricing/yields from your terminal or upload CSV exports.

Can I see the code?

Yes — github.com/Whatsonyourmind/modelforge. Proprietary license, source-available. Commercial use under paid tier or via project-based engagement.
